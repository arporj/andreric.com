# Design System: andreric.com
**Project ID:** 10684813518998808149

## 1. Visual Theme & Atmosphere
Professional, clean, and highly modern. The atmosphere is "Premium Minimalist", using generous whitespace, subtle borders, and an elegant layout to highlight content without distractions. It employs soft glassmorphic effects for elevation and a dynamic feel to reward user interaction.

## 2. Color Palette & Roles
- **Base White** (#ffffff): Used for the main background.
- **Deep Slate/Near Black** (#0f172a): Used for primary text headings and strong emphasis.
- **Slate Gray** (#475569): Used for secondary body text.
- **Accent Royal Blue** (#2563eb): Used for primary call-to-action buttons, active states, and key highlights.
- **Soft Cloud** (#f8fafc): Used for subtle card backgrounds and alternating section backgrounds.
- **Subtle Border** (#e2e8f0): Used for fine lines, glassmorphism borders, and dividers.

## 3. Typography Rules
- **Primary Font:** 'Inter', sans-serif
- Headings are bold (font-weight: 700), tightly spaced to appear modern and punchy.
- Body text is regular (font-weight: 400) with a relaxed line-height (e.g., 1.6) for optimal readability.

## 4. Component Stylings
* **Buttons:** 
  - Pill-shaped or slightly rounded (rounded-lg or rounded-full).
  - Solid Royal Blue background with white text for primary actions.
  - Hover states should slightly lift the button (shadow-md) and darken the blue.
* **Cards/Containers:** 
  - Subtly rounded corners (rounded-xl or rounded-2xl).
  - Background is either stark white or Soft Cloud.
  - Borders are whisper-thin, using the Subtle Border color.
  - Hovering over interactive cards (like project showcases) should introduce a soft glow or a deeper, diffused drop shadow.
* **Pills/Tags:**
  - Used for technologies or skills. Highly rounded, very faint blue or gray background with medium text color.

## 5. Layout Principles
- Use max-width containers centered on the screen (e.g., max-w-5xl).
- Emphasize vertical rhythm with large gaps between overarching sections (py-16 or py-24).
- Information density is balanced; allow content to breathe. Use grid layouts for repeating items (e.g., projects or tech stack blocks).
