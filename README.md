# andreric.com

Este é o repositório do site pessoal web / CV de André.

## Tecnologias

- **Framework**: React.js com Vite (TypeScript)
- **Estilos**: Vanilla CSS + TailwindCSS (módulos utilitários para integração rápida com UI design tools)
- **Gerenciamento de Design UI**: Construído com o auxílio do Google Stitch Design (MCP)
- **Ícones**: Lucide React
- **Roteamento**: React Router v6+

## Scripts Úteis

- `npm run dev` - Inicializa o servidor de desenvolvimento.
- `npm run build` - Roda o build de produção.
- `npm run preview` - Serve o build localmente.
